MagicPage

Description

A light, responsive & optimized web page for templating/rapid development.
The page refreshes itself every 3 minutes.
This can be adjusted by changing the content attribute value in the appropriate meta tag.

Technology

Hypertext Markup Language(HTML) / Cascading Style Sheets(CSS) / Javascript
HTML for markup, CSS for styling &
Javascript for a little 'Hello World!' action.
